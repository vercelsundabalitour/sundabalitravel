## üöÄ Deployment Steps

### Option 1: Deploy to Vercel (Recommended)

1. **Push to GitHub**

   ```bash
   git add .
   git commit -m "SEO optimization complete"
   git push origin main
   ```

2. **Deploy on Vercel**

   - Go to [vercel.com](https://vercel.com)
   - Import your GitHub repository
   - Vercel will auto-detect Next.js
   - Click "Deploy"
   - Your site will be live at: `https://your-project.vercel.app`

3. **Add Custom Domain**
   - In Vercel dashboard: Settings ‚Üí Domains
   - Add your domain: `sundabalitour.com`
   - Follow DNS configuration steps
   - SSL certificate is automatic

### Option 2: Deploy to Netlify

1. **Build Command**: `npm run build`
2. **Publish Directory**: `.next`
3. Add `netlify.toml`:

   ```toml
   [build]
     command = "npm run build"
     publish = ".next"

   [[redirects]]
     from = "/*"
     to = "/index.html"
     status = 200
   ```

### Option 3: Deploy to Your Own Server

1. **Build the project**:

   ```bash
   npm run build
   ```

2. **Start production server**:

   ```bash
   npm start
   ```

3. **Use PM2 for process management**:
   ```bash
   npm install -g pm2
   pm2 start npm --name "sundabalitour" -- start
   pm2 save
   pm2 startup
   ```

## üìä Post-Deployment SEO Tasks

### 1. Submit to Search Engines

**Google Search Console**

- Go to [search.google.com/search-console](https://search.google.com/search-console)
- Add property: `https://sundabalitour.com`
- Verify ownership (HTML tag method)
- Submit sitemap: `https://sundabalitour.com/sitemap.xml`
- Request indexing for main pages

**Bing Webmaster Tools**

- Go to [bing.com/webmasters](https://www.bing.com/webmasters)
- Add site and verify
- Submit sitemap

### 2. Add Verification Codes

After getting verification codes, add them to `app/layout.tsx`:

```typescript
verification: {
  google: 'your-google-verification-code',
  bing: 'your-bing-verification-code',
}
```

### 3. Set Up Analytics

**Google Analytics 4**

```bash
npm install @next/third-parties
```

Add to `app/layout.tsx`:

```typescript
import { GoogleAnalytics } from "@next/third-parties/google";

// In your component
<GoogleAnalytics gaId="G-XXXXXXXXXX" />;
```

### 4. Social Media Integration

Update Open Graph image:

- Create a 1200x630px image with your branding
- Save as `public/og-image.jpg`
- Reference in metadata

### 5. Performance Optimization

**Check Performance**

- Run Lighthouse audit in Chrome DevTools
- Check [PageSpeed Insights](https://pagespeed.web.dev/)
- Aim for 90+ scores

**Optimize if needed**:

- Compress images further
- Enable caching headers
- Consider CDN for images
